Arpspoof
=========

A simple arpspoofing program written in python

<br>
<br>
#Usage
```sh
python arpspoof.py -t <target_ip>
```
<br>
* If no target is specified a list of live IP's will be displayed.
<br>
<br>

#### Dependencies
* python 2.7
* scapy
* signal
